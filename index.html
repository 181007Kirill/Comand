Работа в консоли

is - вывести список содержимого папки, в которой находимся

cd - сменить папку

mkdir - команда для создания новых каталогов

rmdir - удаляет каталог из файловой системы

rm-rf - команда выполняющая рекурсивное удаление без подтверждения

cd - команда командной строки для изменения рабочего каталога

touch index.html style.css - команда для создания новых файлов

code index.html style.css - тоже самое, что и touch только сразу открывает их
